[#_error_type]
=== `gf::Error<E>`

[source]
----
#include <gf2/Result.h>
template<typename E>
struct Error;
----

An `Error<E>` is a simple wrapper around an error type `E`.

See also: <<_error>>, <<_result>>

[source]
----
