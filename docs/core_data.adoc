= `core` Data
v0.1
include::bits/attributes.adoc[]
:toc: right

The data component provides classes for handling various type of data related to graphics objects and resources:

- animations (<<animation_data>>, <<animation_group_data>>)
- audio sources (<<audio_source_data>>)
- curves (<<curve_data>>, <<curve_group_data>>)
- shapes (<<shape_data>>, <<shape_group_data>>)
- sprites (<<sprite_data>>)
- text (<<text_data>>)

These classes are pure data, they are declarative and are meant to be serialized if necessary.
