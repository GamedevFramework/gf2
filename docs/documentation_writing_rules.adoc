This is the reference for the writing rules of gf2 documentation. If you think something is not well-written, check the rules. If there is no rule, then it may be a good idea to add a new rule to enforce consistency throughout the pages.

== Symbol naming

Where to put `gf::`:

- document title
- section titles
  - types
  - functions
  - variables

Where to _not_ put `gf::`:

- inline text
- member types
- member functions
